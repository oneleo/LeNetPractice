### 1、建議的環境變數（Windows）

* 進到【控制台】→【系統及安全性】→【系統】→【進階系統設定】→【進階】標籤→【環境變數(N)...】→在【%USERNAME% 的使用者變數(U)】新增環境變數。

``` bat
> set GOROOT="C:\Go"
> set GOPATH="%USERPROFILE%\go;%USERPROFILE%\work"
> set GOBIN="%USERPROFILE%\go\bin"
> set PATH="%PATH%;%GOROOT%\bin;%GOBIN%;%USERPROFILE%\work\bin"
```
